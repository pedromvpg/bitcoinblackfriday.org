How to submit

  • Make a Pull Request to /docs/deals.json. If you don't know how, please create an issue with all details in this format and we will add it for you.
  • When making your PR, make sure to add your product image in /docs/images/2021/[YOURCOMPANYNAME]/[COMPANY_NAME-PRODUCT_NAME.jpg] (all lower case)
  • Include the right categories, please keep "featured": false. Featured is a reserved category.
  • Image should be png or jpg with minimun size of 512x512px and maxium 1024x1024px.
  • Code of conduct
    • Don't be difficult.
    • Keep it Safe for Work [SFW].
    • We reserving the right to no accept any listing for any reason without explanation.
    • Although we don't have a limit of count of listings per company we will use our discretion.
    • No pre-orders, you must have the items in stock to ship.
